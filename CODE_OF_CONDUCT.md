# Code of Conduct

## Our Pledge

We want AURA to be a welcoming open-source project for people building local-first AI assistants, voice interfaces, desktop automation, browser integrations, and mobile companion tools.

Everyone participating in this project is expected to act with respect, patience, and professionalism.

## Expected Behavior

Examples of behavior that helps the community:

- Use welcoming and inclusive language
- Respect different experience levels, backgrounds, and viewpoints
- Give constructive feedback focused on the project
- Be clear when proposing changes, reporting bugs, or reviewing code
- Assume good intent while still being careful with security, privacy, and user safety

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, intimidation, discrimination, or personal attacks
- Sexualized language or imagery
- Publishing private information without explicit permission
- Deliberately derailing discussions or abusing issue and pull request threads
- Encouraging unsafe use of local automation, voice capture, screenshots, browser access, or network services

## Enforcement

The maintainer may remove comments, close issues, reject pull requests, block users, or take other reasonable steps to protect the project community.

Project spaces include GitHub issues, pull requests, discussions, documentation, community channels, and any other official AURA collaboration spaces.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer through the GitHub profile linked from this repository. Please include relevant context, links, screenshots, or other details that can help with review.

Reports will be handled with care and confidentiality where possible.

## Attribution

This code of conduct is inspired by common open-source community guidelines and adapted for AURA's local-first AI assistant project.
